# 博客分类与标签体系

## 一、文章分类（Category）

本站文章分为以下 **6 个互斥分类**，每篇文章只归属其一：

| 分类 | 适用内容 | 典型例子 |
| :--- | :--- | :--- |
| **实战攻防** | 真实环境下的渗透测试、红蓝对抗、授权漏洞挖掘（目标为在线真实系统） | 某企业外网打点到域控、HVV实战绕过WAF、SRC漏洞提交记录 |
| **漏洞原理** | 漏洞的成因、代码分析、利用条件与修补方案（核心是解释漏洞本身） | Log4j2 JNDI注入源码分析、PHP类型混淆漏洞详解、SQL注入原理 |
| **挑战记录** | 所有非真实环境的解题/通关/夺旗：CTF、VulnHub、HTB、本地靶场（DVWA、Sqli-labs等） | Vulnhub靶机渗透全记录、CTF决赛Web题WriteUp、DVWA通关到进阶 |
| **工具手册** | 安全工具的安装、配置、使用技巧、高级功能（以工具为中心） | Nmap隐蔽扫描技巧、Burp Suite插件编写、Sqlmap tamper脚本 |
| **运维排错** | 搭建环境、配置服务、管理服务器时遇到的错误及解决方案 | Docker容器端口无法访问、MySQL启动报错1045、Kali无线网卡失效 |
| **基础知识** | 操作系统、网络协议、编程语言、安全理论等底层知识 | TCP三次握手与SYN洪水、Linux权限管理、汇编基础、PKI体系 |

### 分类快速决策树

1. 目标是**真实在线系统**？ → `实战攻防`  
2. 核心是**解释漏洞原理**？ → `漏洞原理`  
3. 内容是**解决故障/错误**？ → `运维排错`  
4. 内容是**工具的使用方法**？ → `工具手册`  
5. 内容是**基础理论**（不涉及具体漏洞/工具）？ → `基础知识`  
6. 其余一切**非真实环境的解题/通关/夺旗** → `挑战记录`
